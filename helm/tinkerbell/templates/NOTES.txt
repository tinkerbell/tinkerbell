ImageVolume feature gate to be enabled on your Kubernetes cluster.

  Kubernetes Version  | ImageVolume | Default

  1.33 - 1.34         | Beta        | false
  1.35+               | Beta        | true

to be created by the API server.

For versions where the default is "false", you must explicitly enable the
feature gate on the kubelet and API server.
